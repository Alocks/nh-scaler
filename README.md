This is an chrome extension that upscales and sharpens image in real time using the GPU and process it in real-time. The fastest one takes 100ms to process, so its almost unnoticeable the processing, and the improvements in the image makes way better to enjoy your reading, even more when you have a 4K monitor like me. Also, features in the extension:

- You can change between 4 profiles (fast, balanced, quality, high quality, max quality)

- Process the next pages before loading into rendered page

- Enhances the images in parallel

- Logs the time it takes to process every image. E.g:


[NH Scaler] process:upscale-time {ts: '2026-05-11T08:45:55.799Z', sourceUrl: 'https://i3.nhentai.net/galleries/3927153/6.webp', page: 6, duration: '124.60ms'}
